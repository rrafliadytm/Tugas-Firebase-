# **App Name**: VerdantDo

## Core Features:

- Google OAuth Login: Secure user authentication via Google OAuth.
- Task Creation: Allow users to add new tasks with descriptions and due dates.
- Task Completion: Enable users to mark tasks as complete.
- Task Categorization: Allows user to categorize tasks to provide higher level organization.
- View Active Tasks: Display a list of all active (incomplete) tasks.
- View Completed Tasks: Display a list of all completed tasks.

## Style Guidelines:

- Primary color: Fresh green (#A7D1AB) to evoke productivity and freshness.
- Background color: Off-white (#F5F5F5) for a clean and modern look.
- Accent color: Light teal (#9AD0EC) for interactive elements and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif combining a modern look with some warmth.
- Use simple, clean icons for task status and actions.
- Clean and intuitive layout with clear separation of tasks and categories.
- Subtle animations for task completion and transitions.
